## Installation

To add the `Tooltip` to your project:

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab value="npm">
    ```bash
    npx @gravcn/cli add tooltip
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm dlx @gravcn/cli add tooltip
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn dlx @gravcn/cli add tooltip
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bunx @gravcn/cli add tooltip
    ```
  </Tab>
</Tabs>


This will automatically pull in the following official Shadcn primitives as dependencies:
*   `tooltip`

## Usage Example

Here is how you can render the `Tooltip` component:

```tsx title="demo/Tooltip.tsx"
import { Button } from "@/components/ui/button";
import {
  Tooltip,
  TooltipContent,
  TooltipProvider,
  TooltipTrigger,
} from "@/components/ui/tooltip";

export function TooltipDemo() {
  return (
    <TooltipProvider>
      <Tooltip>
        <TooltipTrigger asChild>
          <Button variant="outline">Hover me</Button>
        </TooltipTrigger>
        <TooltipContent>
          <p>Add to library</p>
        </TooltipContent>
      </Tooltip>
    </TooltipProvider>
  );
}
```
